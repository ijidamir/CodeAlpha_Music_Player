Code Alpha Music Player
A simple and interactive web-based music player built as part of my internship tasks at Code Alpha. This project focuses on audio handling, dynamic UI updates, and user interaction using JavaScript.
 Features
Play and pause functionality
Next and previous track controls
Audio progress bar with real-time updates
Display of current time and total duration
Dynamic song switching
Responsive and clean UI
Volume control
🛠️ Built With
HTML5 – Structure of the player
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Audio control and logic

⚙️ How It Works
Audio is controlled using the HTML <audio> element.
JavaScript manages:
Play/pause toggle
Switching between songs
Updating progress bar (currentTime & duration)
The Next/Previous buttons update the current track dynamically.
The progress bar reflects playback in real-time.

💡 Learning Objectives
This project helped me understand:
Working with the JavaScript Audio API
Synchronizing UI with media playback
Handling events like play, pause, and ended
Managing multiple media files
